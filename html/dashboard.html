NU grid system

Provides you with 4 different box sizes for a flexible layout. Basically, you can choose if you want one, two or three columns. Throw in a couple of sections and articles and you are good to go.

Simplicity at heart

Checkout the source code. It's as tough to get things wrong as it is simple to get the hang of ! Ready-made css classes can pile up to create the desired effect in no time at all. Our css file also aims for optimum modularity.

Lightweight

The very first concern whilst making this framework was feather-weight files. We strive to provide just what is needed with no unecessary bloating. Our pre-compressed files say it all !

Fits on all screens

Easily control which screen resolutions should be taken care in the css file. You can also activate our fluid grid layout if you feel confortable with it ! Respectively choose admin.css or admin-fluid.css corresponding to your preferred configuration.

Works on all browsers

Administration frameworks are often destined to be used on old systems...and old browsers. NUadmin has been conceived to work on absolutely all browsers with no usage problem. Of course, esthetics may change, but be sure it'll work just fine.

Forms, Inputs and Buttons and…

Data input frenzy ! We have prepared many different display options for all types of inputs. Add a few css classes here and there (or none if you like our 'default' look) which you'll find here to get things running. We even have a few data tables looks ready for news listings and so on. Find out more in the NUadmin wiki.

No Javascript ?

NUadmin's github project has a javascript enabled branch. Indeed, NUadmin works without any JS, but it is greatly enhanced by it. Don't forget to have a look !

Feedback
